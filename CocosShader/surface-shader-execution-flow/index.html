执行流程优化

  1. 合理安排 Pass 顺序：深度预Pass → 不透明物体 → 天空盒 → 透明物体
  2. 批处理优化：相同材质的物体合并渲染
  3. 状态变更最小化：减少渲染状态切换次数
  4. early-Z 优化：利用深度测试减少 overdraw
