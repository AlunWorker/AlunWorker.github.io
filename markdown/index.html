MarkDown学习

发表于2024-06-19|更新于2024-06-19|MarkDown
|字数总计:600|阅读时长:1分钟|阅读量:

Markdown 是一种轻量级的标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成结构化的 HTML 页面。它的目标是实现“易读易写”，并且具有一定的可读性，即使在没有格式转换的情况下也能被阅读。

以下是 Markdown 的一些基本语法：

标题：使用井号 # 表示不同级别的标题。

强调：使用星号 * 或下划线 _ 来强调文本。

这是斜体文本
这也是斜体文本
这是加粗文本
这也是加粗文本

列表：

  1. 无序列表使用星号 *、加号 + 或减号 - 来创建。
    • 列表项一
    • 列表项二
    • 列表项三
  2. 有序列表使用数字后跟一个点来创建。
    1. 第一项
    2. 第二项
    3. 第三项

链接：使用方括号和圆括号来创建链接。

这是一个链接

图片：使用感叹号和方括号来创建图片。

代码：使用反引号来创建行内代码，或者使用三个反引号来创建一个代码块。

引用：使用大于号 > 来创建引用文本。

这是一段引用文本

水平线：使用三个星号 *** 或三个减号 --- 来创建水平线。

表格：使用管道符 | 和连字符 - 来创建表格。

标题1标题2标题3
单元格1单元格2单元格3
单元格4单元格5单元格6

换行显示：加两个空格。

这是第一行。
这是第二行。
